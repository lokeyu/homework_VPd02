# Описание изменений: консольный конвертер валют

В рамках домашнего задания был разработан и протестирован консольный интерфейс для конвертации валют в файле `currency.py`.

## Внесенные изменения

- Создано интерактивное консольное меню, позволяющее пользователю:
  1. Выполнить конвертацию валют.
  2. Посмотреть список всех поддерживаемых кодов валют.
  3. Обновить курсы валют из интернета вручную.
  4. Выйти из программы.

- Реализован парсинг JSON для чтения курсов валют из файла `currency_rate.json`:
  - поддерживается плоская структура JSON — один ответ API;
  - поддерживается вложенная структура JSON — несколько избранных базовых валют;
  - обрабатываются сценарии отсутствия или повреждения файла.

- Реализован расчет кросс-курса:
  - кросс-курс между любыми двумя валютами вычисляется по формуле `target_rate / source_rate`;
  - это позволяет конвертировать не только из базовой валюты, но и между любыми валютами из списка доступных.

- Добавлена валидация пользовательского ввода:
  - коды валют проверяются по списку доступных в файле;
  - сумма конвертации проверяется на число;
  - поддерживается запятая и точка в качестве десятичного разделителя;
  - исключаются отрицательные значения;
  - поддерживается корректная обработка Ctrl+C.

## Верификация и тестирование

- Создан тестовый модуль `test_currency.py`, проверяющий:
  - извлечение курсов из плоских структур JSON;
  - извлечение курсов из вложенных структур JSON;
  - математику расчета кросс-курсов.
- Тесты запущены успешно.
- Проверена компиляция `currency.py` без ошибок.
- Файл `currency_rate.json` инициализирован курсами валют.

## Что получилось

В результате получилась CLI-программа, которая берет данные из JSON-файла, умеет обновлять курсы через API и конвертировать сумму между валютами с расчетом кросс-курса.
