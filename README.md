# Конвертер валют (CLI)

Домашнее задание VPd02: работа с документацией API, JSON-форматом данных, локальным кэшированием ответа API и консольным интерфейсом для конвертации валют.

## Что умеет программа

- получает курсы валют через открытый API `https://open.er-api.com/v6/latest/<currency>`;
- сохраняет полученные данные в файл `currency_rate.json`;
- читает курсы из локального JSON-файла;
- показывает список доступных валют;
- конвертирует сумму между двумя валютами;
- рассчитывает кросс-курс по формуле `target_rate / source_rate`;
- проверяет корректность ввода валюты и суммы;
- умеет обновлять курсы вручную из меню.

## Файлы проекта

- `currency.py` — основной файл программы.
- `currency_rate.json` — локальный файл с кэшированными курсами валют.
- `test_currency.py` — простые тесты для проверки извлечения курсов и расчета кросс-курсов.
- `walkthrough.md` — описание проделанной работы и проверки.
- `requirements.txt` — зависимости проекта.
- `.gitignore` — исключения для Git.

## Установка и запуск

```bash
python -m venv .venv
```

Windows PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

Linux / macOS:

```bash
source .venv/bin/activate
```

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск программы:

```bash
python currency.py
```

## Запуск тестов

```bash
python test_currency.py
```

## Пример сценария

1. Запустить `python currency.py`.
2. Выбрать пункт `1. Конвертация валюты`.
3. Ввести исходную валюту, например `USD`.
4. Ввести целевую валюту, например `RUB`.
5. Ввести сумму.
6. Получить результат конвертации и текущий курс.
